## 📖 概要 (Overview)
駅ナカの書店で働く店員さんを想定した、実践的な英会話学習アプリです。
お客様からの質問（音声）を聞き取り、バラバラになった単語を正しい語順で並べ替えて返答を作成します。

### 特徴
*   **実践的なシナリオ**: 「コミック売り場はどこ？」「免税はできますか？」など、現場で即座に使えるフレーズを厳選。
*   **音声リスニング＆スピーキング**: 質問の読み上げに加え、自分が作った文章もネイティブ発音で読み上げられます（スマホ完全対応）。
*   **ゲーム要素**:
    *   **タイムアタック**: 素早く回答すると高得点（タイムボーナス）。
    *   **リトライ機能**: 間違えた問題は最後に再出題され、完璧になるまで終われません。
    *   **ランク判定**: スコアに応じて称号が変わります。

## 🎮 遊び方 (How to Play)
1.  **START** ボタンを押してゲーム開始！
2.  お客様の質問（英語音声＋日本語字幕）が表示されます。
3.  画面下の単語リストから、正しい返答になるように順番に単語をタップします。
4.  **CHECK** ボタンで答え合わせ！
5.  全10問正解を目指して高スコアを狙おう！

## 📱 スマホでの動作について
*   **iPhone / Android 対応**: ブラウザ（Safari, Chrome）での動作を確認済みです。
*   **音声トラブル対策済み**: スマホ特有の「音が鳴らない」「途切れる」問題に対し、独自の対策コードを実装しています。
    *   *※ マナーモード時は音が鳴らない場合があります。解除してお楽しみください。*

## 🛠 技術スタック (Tech Stack)
*   **HTML5 / CSS3**: モダンなレスポンシブデザイン（Glassmorphism採用）。
*   **Vanilla JavaScript**: フレームワークを使わず、軽量・高速な動作を実現。
*   **Web Speech API**: ブラウザ標準の音声合成エンジンを使用（多言語・異機種対応チューニング済み）。
