## 🚀 モード紹介 (Modes)

### 🔰 TRAINEE (初級編)
**「まずはここから！スピード勝負」**
*   お客様の質問に対して、正しい返答を3つの選択肢から選ぶクイズ形式。
*   瞬発力と基本フレーズの暗記に最適です。
*   目標：新人 (Newbie) から 店長 (Store Manager) への昇格！

### 😎 MANAGER (中級編)
**「実践トレーニング！文法組み立て」**
*   バラバラになった単語を正しい語順で並べ替えて返答を作ります。
*   ただ暗記するだけでなく、「英語を組み立てる力」を養います。
*   間違えた問題はリトライ機能で確実に習得。

---

## ✨ 主な機能 (Features)
*   **実践的シナリオ**: 「免税」「場所案内」「在庫確認」など、書店や小売店で本当によく使うフレーズを厳選。
*   **スマホ完全対応**:
    *   移動中や休憩時間にも片手でプレイ可能。
    *   **音声トラブル対策済み**: スマホのブラウザでも音声が途切れたり無音にならないよう、独自のオーディオ制御エンジンを搭載。
*   **ネイティブ発音**: 質問も回答も、ネイティブに近い発音で読み上げられます（リスニング強化）。

## 🛠 技術スタック (Tech Stack)
*   **Frontend**: HTML5, CSS3 (Glassmorphism UI), Vanilla JavaScript
*   **Audio**: Web Speech API (with Custom Audio Manager for mobile stability)
